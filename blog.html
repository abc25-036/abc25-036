My Blog

Here I share my learning journey, coding tips, and reflections as a web developer student.

Challenge 1: My First Website

One of my biggest challenges was building my very first website. I struggled with structuring pages, linking CSS correctly, and keeping my code organized. This taught me the importance of folder structure and clean HTML.

Challenge 2: Using Pictures

I struggled with using images properly. At first, my pictures wouldn’t load because I didn’t understand file paths. Later, I learned to keep all images inside an images/ folder and reference them correctly in my HTML.

Challenge 3: Confidence in Presentation

Another challenge was presenting my work. I often felt nervous and doubted my skills. Writing this blog helps me reflect and build confidence in explaining my projects step by step.

Lessons Learned

  • Organizing files and folders makes coding easier and prevents broken links.
  • Using the <img> tag correctly and keeping images in one folder solved my picture struggles.
  • Confidence grows when I practice explaining my work and reflect on my progress.
  • Challenges are part of learning — each mistake teaches me something valuable.
